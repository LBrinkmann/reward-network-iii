# Python App Template

Python version >= 3.7 is required.

## Setup

```
python3 -m venv .venv
. .venv/bin/activate
pip install --upgrade pip
pip install wheel
pip install -e ".[dev]"
```

## Run Locally

```
env FLASK_APP=app/app.py FLASK_ENV=development flask run
```
